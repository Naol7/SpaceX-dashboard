## Getting Started

1. Clone repository
```bash
git clone https://github.com/Naol7/SpaceX-dashboard.git
cd SpaceX-dashboard
```
2. Install dependencies
```bash
npm install
```
3. Start development server
```bash
 npm run dev
 ```
4. Open in browser
   Navigate to http://localhost:5173
